 # Seed Moth Detection (Proof of Concept)
This repository contains a proof-of-concept for detecting moths in pheromone trap images using YOLO26n model. The goal is to demonstrate a starting point under limited labeled data conditions.

# Repository Structure
- data_analysis.py – script for dataset inspection and visualization
- labels_convertion.py – converts annotations to YOLO format
- train_test_split.py – splits dataset into train/val/test
- tuning_script.py – script for hyperparameter tuning (Ray Tune)
- train_yolo.py – training script for YOLO
- predict.py – script for inference on test image samples from the Stenoma catenifer class
